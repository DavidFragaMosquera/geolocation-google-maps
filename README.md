# geolocation-google-maps

```
Utilizando la api de Google Maps para geolocalización e inserción de mapas en proyectos Vue
https://developers.google.com/maps/documentation/javascript/overview
```

## Funcionalidades
```
- Obtener coordenadas (longitud+latitud) de los usuarios siempre que estos nos permiten el acceso a su ubicación.
- Manejar error en caso de que el usuario no apruebe los servicios de ubicación.
- Inserción de google maps totalmente personalizable, con zoom y centrado correcto
```

## Pasos a seguir
```
npm install
npm run serve
insertar API_KEY de nuestro usuario de google cloud en el archivo 'main.js'
